# 🪔 Diwali Sales Analysis Using Python

## 📌 Project Overview

This project focuses on analyzing **Diwali sales data** using Python to uncover customer purchasing behavior, sales trends, and business insights.

The analysis was performed using:
- **Pandas** for data cleaning and manipulation
- **NumPy** for numerical operations
- **Matplotlib** for data visualization
- **Seaborn** for advanced statistical visualizations

The project aims to help businesses understand:
- Customer demographics
- Sales performance
- Product demand
- State-wise sales trends
- Occupation-based purchasing patterns
- Gender and marital status impact on purchasing behavior

# 📈 Exploratory Data Analysis (EDA)

## ✅ Gender Analysis

Analyzed:
- Total number of male and female buyers
- Purchasing power by gender

### Key Insight:
- Female customers contributed more to overall sales.

---

## ✅ Age Group Analysis

Analyzed:
- Customer distribution across age groups
- Spending patterns by age

### Key Insight:
- Customers aged **26–35 years** made the highest purchases.

---

## ✅ State-wise Sales Analysis

Analyzed:
- Top states by total orders
- Top states by revenue generation

### Key Insight:
- Uttar Pradesh, Maharashtra, and Karnataka generated the highest sales.

---

## ✅ Marital Status Analysis

Analyzed:
- Buying behavior based on marital status
- Gender-wise marital purchasing patterns

### Key Insight:
- Married women showed higher purchasing power.

---

## ✅ Occupation Analysis

Analyzed:
- Customer occupation distribution
- Revenue contribution by occupation

### Key Insight:
- Customers working in IT, Healthcare, and Aviation sectors contributed significantly to sales.

---

## ✅ Product Category Analysis

Analyzed:
- Most purchased product categories
- Revenue generated by categories

### Key Insight:
- Food, Clothing, and Electronics categories performed strongly.
